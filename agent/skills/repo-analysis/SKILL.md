# Repo Analysis Skill

This skill enables the AI agent to analyze a GitHub repository
and provide meaningful insights.

## Capabilities

- Explain the purpose of a repository
- Describe the project structure
- Identify key components
- Detect potential bugs or issues
- Suggest improvements

## Usage

The user provides a GitHub repository URL.

Example:

https://github.com/user/repository

The agent analyzes the repository and generates insights for developers.
